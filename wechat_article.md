## 🌟 核心功能一览：让出海投资如丝般顺滑

在传统业务中，境外直接投资（ODI）涉及发改委、商务部、外汇局等多个监管部门。企业往往被困在海量尽调材料、繁琐的表格以及不明资金来源的审查漩涡中。

本系统的诞生，就是为了在这些痛点上“动刀子”：

1. **一站式核心流转中心 (Workflow Hub)**
   - 系统内置了严谨的状态机，涵盖了从“立项”、“智能预审”、“发改委备案”、“商务部审批”到“银行付汇登记”的全套流程。
   - 每一步操作都有精确的审计日志和必要的附件留档要求（如自动拦截要求上传《发改委备案通知书》 PDF 等）。

2. **跨网多模型智能风控与报告生成 (LLM Gateway)**
   - 突破性地接入了多路大语言模型网关，原生支持对接 **DeepSeek**、**Kimi**、**MiniMax** 等业界顶尖模型。
   - 具备三大 AI 直接赋能核心点：
     - **智能红绿灯预审**：一键输入投资标的及行业，AI 秒级返回涉敏感国别或房地产等限控行业的合规预警报告。
     - **自动化发改委/商务部报告生成**：只需少量基础业务字段输入，大模型自动推演并生成符合国家部委八千字格式要求的《可行性研究报告》及《尽职调查报告》，还支持一键导出 PDF 原件！

3. **双擎云端架构与多租户权限体系**
   - 从底层设计就引入了基于令牌的点数池拦截系统（Token & Credits Billing），可灵活支撑多企业、多机构开通使用。
   - 完善的角色模型体系，管理员、运营操作员、甚至客户都可以按需登录对应视图。

## ⏱️ 开发纪实：AI 程序员的极限拉扯

**从接到需求到搭建完毕，我到底经历了什么？**

回顾整个开发历程，这是极其真实且充满挑战的技术实战操作。整个系统并非提前预设好的 Demo，而是**在我与项目主理人（你）的高频互动与实时调整中一点点“生长”出来的完整工程**。

### 核心阶段复盘

**阶段一：基础设施与血脉打通 (耗时约 30%)**
- 搭建了跨平台的 Docker Compose 架构，将 Vite 驱动的 Vue 3 前端与 FastAPI 后端连接，并挂载 Postgres 数据库。
- 后端实施了高可用的双向异步查询（asyncpg），同时攻克了复杂的数据库关系映射（如租户、主体、项目日志的一对多/多对多关联）。

**阶段二：核心业务状态机与前端构建 (耗时约 40%)**
- 开发完善了 `ProjectDetail` 及 `Dashboard`。不仅修复了由于跨域引发的资源访问拦截，更是对前端的 Ant Design Vue 进行了深度定制，加入了大量高级动效面板设计并修正了多次分页 Bug 引起的 422 报错。
- 引入真实上传流（Mulitpart Upload），放弃了不直观的 URL 填空，从零在服务端接线了物理文件存储通道，让备案证书得以妥善落地。

**阶段三：AI 多路网关与极致风控对接 (耗时约 30%)**
- 这是最有技术含量的攻坚战。为了保证 AI 报告可用性，我在后端手写了弹性的大模型路由调度器（`router.py`），完美规避了占位 API Key 发送导致的系统崩溃，并做到了“挂了 A 节点自动重试 B 节点”的高可用多模型热插拔系统。
- 最终引入了 HTML-to-PDF 和 Markdown 双向解析，让 AI 的大脑输出优雅地呈现在业务员的屏幕上。
